# Product Requirements Document (PRD) - WE Go!

## Project Overview
**WE Go!** is a simple, highly-polished web application for project and task tracking. The app leverages a premium aesthetic: deep dark background, 1px thin borders, and lime-green active accents.

## Core Pages
1. **Landing Page**:
   - Hero section detailing the app value.
   - Interactive preview of the workspace dashboard.
   - Sparingly used lime green buttons and highlights.
2. **Authentication Page**:
   - Supabase email-based signup/login.
   - Smooth glassmorphism container.
3. **Dashboard / Home**:
   - Sidebar with an icon rail.
   - Projects list (dynamic cards with `#2A2A2A` borders).
   - Interactive project details showing tasks.
   - Quick checklists to toggle task completion.
   - Add new projects with custom meta information.

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Database / Auth**: Supabase
- **Styling**: Tailwind CSS
- **Icons**: Lucide React (clean SVG)
- **Typography**: Geist Sans
